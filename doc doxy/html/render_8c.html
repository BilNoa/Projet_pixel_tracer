void draw_layer_shapes ( Area *  area,
Layer *  layer

Convertit les shapes d'un layer en pixels et les écrit dans la grille.

Pour chaque shape du layer, génère la liste de pixels via create_shape_to_pixel() puis marque les cellules correspondantes à FULL_CHAR dans area->area[][]. Les pixels hors bornes sont ignorés.
